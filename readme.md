1. Инициализация проекта
   ```npm init -y ```

2. Установка пакетов
   ```npm i -D nodemon cross-env```

3. Запуск демона
   ```npm run dev```

4. Установка телеграф и конфиг для чтения из default.json config
   ```npm i telegraf config```

5. Пакет для скачивания файлов
   ```npm i axios fluent-ffmpeg @ffmpeg-installer/ffmpeg```

6. Установить openai
   ```npm i openai```



   npm run dev - подключаются дев ключи
   npm start - подключаются прод ключи
